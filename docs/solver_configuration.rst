Solver configuration
====================

.. note::
   ``ember.config.SolverConfig`` (documented on this page) belonged to the
   legacy ``ember.run`` solver loop and has been removed. The current
   solver's configuration is :class:`ember.scree.ScreeConfig`.

The :doc:`algorithm` page describes how each parameter below influenced
the stages of the legacy time-stepping loop.
